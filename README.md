# Grow a Garden — Plant, Pet & Mutation Reference

A community reference for Roblox's **Grow a Garden** and **Grow a Garden 2** — plant base values, mutation multipliers, pet stats, and current event codes. Built for players who want quick lookups without digging through wikis or Discord threads.

## What's in here

- Plant base Sheckle values (GAG 1 & GAG 2)
- Mutation multiplier list (Gold, Rainbow, Tidal, and more)
- Pet ability reference
- Active redeem codes, updated as they drop

## Live tools

Static reference data is useful, but for anything that needs live calculation — stacking multiple mutations, checking a trade, comparing crop profit per hour — use the calculators instead of doing the math by hand:

- [Plant Value Calculator](https://mygardencalculator.com/) — 324 plants, 167 mutations, GAG 1 & GAG 2
- [Grow a Garden 2 Calculator](https://mygardencalculator.com/grow-a-garden-2/) — GAG2-specific values and weather modifiers
- [Trade Calculator](https://mygardencalculator.com/grow-a-garden-trade-calculator/) — instant WIN / FAIR / LOSS trade check
- [Pet Calculator](https://mygardencalculator.com/grow-a-garden-pet-calculator/) — pet XP, age, and weight tracking
- [Egg Hatch Calculator](https://mygardencalculator.com/grow-a-garden-egg-hatch-calculator/) — hatch time for every egg
- [Profit Calculator](https://mygardencalculator.com/grow-a-garden-profit-calculator/) — rank crops by real Sheckles-per-hour
- [Current Codes](https://mygardencalculator.com/#codes) — active GAG 1 & GAG 2 codes
